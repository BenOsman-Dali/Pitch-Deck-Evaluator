SETUP INSTRUCTIONS:

1. Install Python 3.8+
2. Install dependencies:
   pip install streamlit pypdf groq
3. Get a free API key from: https://console.groq.com
4. Run the app:
   streamlit run app.py
5. Enter your Groq API key in the sidebar
6. Upload a PDF or paste text
7. Click "Evaluate My Pitch Deck"

---

TECH STACK:
- Python 3.14
- Streamlit (Web Framework)
- PyPDF2 (PDF Text Extraction)
- Groq API + LLaMA 3.3 70B (AI Inference)

---

KEY FEATURES:
- 3 Evaluation Dimensions: Clarity, Storytelling, Problem-Solution
- Scores 0-10 for each dimension
- Top 3 Strengths, Weaknesses, Investor Risks
- Investor Thesis & Overall Verdict
- Downloadable evaluation report

---

KNOWN LIMITATIONS:
- Image-based PDFs require manual text paste (OCR not yet integrated)
- No slide-by-slide analysis yet
- 8,000 character limit per evaluation
